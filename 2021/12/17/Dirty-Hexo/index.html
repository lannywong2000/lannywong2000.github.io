Dirty Hexo

Hexo is a cool way to set up your own blog on github despite its baffling configuration. Ready to get your hands dirty?

A. Prerequisites

  1. Git
    To check, run git -v in Git Bash
  2. Node.js
    To check, run node -v in Git Bash
  3. Markdown Knowledge

B. Hexo Configuration

Run the following commands in Git Bash.

  1. Install Hexo. Check using hexo -v
  2. Install the deployer github.
  3. This allow you to use Markdown syntax and relative address to add image to your blog.

C. Set Up Your Blog

  1. Create a folder for your blog, say gitblog.
  2. cd into gitblog.
  3. Initialize a hexo environment using hexo init.
  4. Edit _config.yml in gitblog.
  5. Generate static webpages.
  6. Local preview.
  7. Deploy.
  8. You should now be able to access your blog online on https://{yourusername}.github.io.
